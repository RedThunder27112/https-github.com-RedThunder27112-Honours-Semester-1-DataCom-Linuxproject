Install Instructions

This install instructions are meant for Linux, OpenSuse. However, the program should be able to run on other Linux Distros, with slight modifications to the install process.

Client

The client is a python .php file. This requires the installation of python (python3) in order to run. It also has a clientDownloads folder which we will explain in more detail in the Running Instructions section. The file is called client2.py, and makes use of the following libraries:

  1. socket (comes with python)
  2. tqdm
  3. os
  4. tkinter
  5. time

In order to run the client2.py file, the following should be done: This will make use of terminal commands:

  1. Install python with terminal command, “sudo zypper install python3”
  2. Install tqdm for progress bars, “pip install tqdm”
  3. Install tkinter for client GUI, “sudo zypper install python3-tk”
  4. cd “location/of/client2.py
  5. Ensure clientDownloads is in same directory as client2.py
  6. Run client “python3 client2.py”

Server

The server has a few files and a folder directory in order to run:

  1. server2.py is a python script, and is the main server which is run. This script will activate and control the below files and scripts.
  2. googleD.sh is a bash script, which is in charge of sending a backup to google drive.
  3. mediaPlayer.sh is a bash script, which is in charge of playing the music playlists.
  4. morseCodeSound.py is a python script, which is in charge of converting text to morse code, and playing the morse code via the servers speakers.
  5. passwords.txt is a file that stores the passwords stored on the server.
  6. dir2 folder, stores all of the music playlist folders eg ./dir2/playlist1

server2.py installation needs:

The file makes use of the following libraries:

  1. socket (comes with python)
  2. tqdm
  3. os
  4. threading
  5. time

In order to run, this will make use of terminal commands:

  1. Install python with terminal command, “sudo zypper install python3”
  2. Install tqdm for progress bars, “pip install tqdm”
  3. cd “location/of/server2.py
  4. Ensure the dir2 folder is in same directory as server2.py
  5. Ensure passwords.txt is in same directory as server2.py
  6. Ensure relevent installations for googleD.sh, mediaPlayer.sh and morseCodeSound.py are installed, and they are in same directory as server2.py
  7. The firewall needs to be opened on port 5001 to allow client connections, “sudo firewall-cmd –zone=public –add-port=5001/tcp –permanent”
  8. Run server “python3 server2.py”

googleD.sh installation needs:

In order to run, this will make use of terminal commands:

  1. Create or use your existing account from https://cloud.google.com
  2. Create or use an existing bucket storage
  3. Install the google drive SDK from https://cloud.google.com/sdk/docs/install
  4. Edit the directory variable “FOLDER_TO_BACKUP=desired/backup/directory” in googleD.sh
  5. Edit the bucket name variable “BUCKET_NAME=nameOfStorageBucket” in googleD.sh
  6. Install google Drive SDK “pip install google-api-python-client”

mediaPlayer.sh installation needs:

In order to run, this will make use of terminal commands:

  1. Install mpv, “sudo zypper install mpv”

morseCodeSound.py installation needs:

The file makes use of the following libraries:

  1. sys
  2. subprocess
  3. time
